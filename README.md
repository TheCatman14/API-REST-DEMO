API REST DEMO

Demo di API REST back-end in Java con Spring Boot e database SQL.
Questa repository contiene un backend Spring Boot che espone endpoint REST per operazioni CRUD, con logica divisa in controller, service e repository e persistenza su database SQL.

Cosa fa il progetto?
	-Implementa un’API backend RESTful in Java con Spring Boot
	-Struttura MVC e persistenza con JPA
	-Connessione ad un database SQL reale
	-Esegue operazioni CRUD tramite HTTP standard
	-Usa convenzioni Spring per configurazione e gestione delle entità
					
Requisiti:
	-Java JDK 25
	-Maven per la gestione delle dipendenze
	-Un database SQL (MySQL)
	-IDE (io ho utilizzato Eclipse)
	
Impostazioni Database:
	-MySQL come Database Management System (DBMS)
	-L'applicazione è impostata per funzionare su un host locale (localhost)
	-Porta 3306
	-Nome database "myDb"
	-Username "root"
	-Password "Cat-SQL"
